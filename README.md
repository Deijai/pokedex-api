# Executar em dev
1. Clonar o repositorio
2. Executar

```
npm i

```

3. Ter o Nest CLI instalado

```
npm i -g @nestjs/cli
```
4. Subir a base de dados

```
docker-compose up -d
```


## Stack utilizada
* MongoDB
* Nest
* Docker
